# Progetto per esame VISIONE COMPUTERIZZATA E SISTEMI COGNITIVI nel corso di Laurea Magistrale LM-32 - Ingegneria Informatica e dell’Intelligenza Artificiale Applicata

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![PyTorch](https://img.shields.io/badge/PyTorch-2.0+-ee4c2c.svg)](https://pytorch.org/)

Questo progetto analizza il dualismo tra **prestazioni** e **sicurezza** nelle reti neurali convoluzionali (CNN). L'obiettivo è sviluppare un modello ad alta precisione sul dataset **CIFAR-10**, interpretarne le decisioni tramite tecniche di **Explainable AI (Grad-CAM)** e valutarne la vulnerabilità ad **attacchi adversarial** di tipo White-Box e Black-Box.
